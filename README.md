# Karsten & Kuiper — Logo App

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo → Vercel detects Next.js automatically
4. Click **Deploy**

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
